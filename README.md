# Spam Email Detector 

A Machine Learning project that classifies emails as Spam or Not Spam using Naive Bayes algorithm.

## About the Project
This project automatically detects spam emails using Python and Scikit-learn. 
The model was trained on email data and achieved 100% accuracy.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Google Colab

## How it Works
1. Email text is taken as input
2. Words are converted into numbers using CountVectorizer
3. Naive Bayes model predicts Spam or Not Spam
4. Result is displayed as output

- Not Spam correctly detected ✅
